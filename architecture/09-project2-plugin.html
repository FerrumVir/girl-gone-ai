09 — Project 2 Plug-in Plan

Agent Skill Packs Subscription — stack reuse analysis

Reuse Assessment

The claim of 95%+ stack overlap is optimistic. Realistic estimate: ~85%. While infrastructure components (hosting, payment rails, agent platform, CI/CD, ad pixels) are nearly 100% reusable, the core product mechanics differ substantially. Subscription billing, technical content creation, skill pack delivery, and audience targeting are all net-new workstreams that require dedicated design and implementation.

Key Differences

  • Revenue model: one-time → recurring subscription
  • Content type: lifestyle/productivity → technical/agent workflows
  • Delivery: static downloads → ongoing skill pack updates
  • Audience: general consumers → developers/AI practitioners

What Can Be Reused As-Is

  • Paperclip agent platform and G2 model
  • GitHub Pages / GitHub Actions deploy pipeline
  • Gumroad payment rails (supports subscriptions natively)
  • Meta/Google ad infrastructure (pixels already installed)
  • Banker Agent financial tracking
Project 2 launch is blocked until Project 1 achieves seed capital repayment + 25% ROI threshold. Investor Agent (not yet hired) triggers the transition.
