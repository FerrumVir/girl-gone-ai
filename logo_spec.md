# Logo Specifications

## 1. Format Requirements
- **Vector Format:** `.svg` (Primary for all digital use)
- **Raster Formats:** `.png` (Transparent background, high resolution)
- **Favicon:** `.ico` or `.png` (Optimized for browser tabs)

## 2. Visual Elements
- **Icon/Mark:** A distinct symbol that can stand alone.
- **Wordmark:** The brand name styled uniquely.
- **Combination Mark:** The icon and wordmark used together.

## 3. Usage Guidelines
- **Minimum Size:** Ensure legibility at 32x32px.
- **Clear Space:** Maintain a safe buffer zone around the logo.
- **Background Compatibility:** Provide versions for light, dark, and patterned backgrounds.
