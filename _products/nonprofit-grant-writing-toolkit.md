# Nonprofit Grant Writing Toolkit — Google Sheets Template

> Make a copy of this spreadsheet to start using it. A complete grant writing system covering prospect research, deadline tracking, budget templates, narrative frameworks, outcomes measurement, and follow-up management.

---

> **SETUP GUIDE — Get Running in 20 Minutes**
>
> 1. Create a new Google Sheets document
> 2. Create 6 tabs/sheets and name them: Prospect Tracker, Grant Calendar, Budget Template, Narrative Templates, Outcomes Framework, Follow-Up Tracker
> 3. Copy each section below into its corresponding sheet
> 4. Enter all formulas as documented (marked with `FORMULA:`)
> 5. Start by entering 10-15 potential funders into the Prospect Tracker
> 6. Prioritize using the compatibility scorecard before investing writing time
>
> **Tip:** Grant writing is a numbers game. Most organizations have a 20-30% win rate. Track everything so you can learn from rejections and improve each cycle. One yes pays for the time spent on several no's.

## Funder Compatibility Scorecard

Rate each potential funder 1-5 on alignment factors before applying:

| Factor | Weight | Funder A | Funder B | Funder C | Funder D | Funder E |
|--------|:------:|:--------:|:--------:|:--------:|:--------:|:--------:|
| Mission alignment | 25% | /5 | /5 | /5 | /5 | /5 |
| Geographic match | 15% | /5 | /5 | /5 | /5 | /5 |
| Population served match | 20% | /5 | /5 | /5 | /5 | /5 |
| Grant size fits our need | 15% | /5 | /5 | /5 | /5 | /5 |
| Previously funded similar orgs | 15% | /5 | /5 | /5 | /5 | /5 |
| Relationship/connection exists | 10% | /5 | /5 | /5 | /5 | /5 |
| **Weighted Score** | **100%** | **/5** | **/5** | **/5** | **/5** | **/5** |
| **Priority Rank** | | | | | | |

`FORMULA (Weighted Score): =SUMPRODUCT(Weights, Scores)`

**Score Guide:**

- 4.0-5.0: Strong fit — apply immediately
- 3.0-3.9: Good fit — apply if capacity allows
- 2.0-2.9: Moderate fit — deprioritize
- Below 2.0: Poor fit — do not pursue

## Budget Justification Narrative Template

For each line item, write 1-3 sentences explaining:

1. What the cost is
2. How it was calculated
3. Why it is necessary for the project

**Example:**
"Program Director (0.5 FTE, 12 months): $35,000. The Program Director will dedicate 50% of their time to managing daily program operations, supervising staff, and ensuring fidelity to the program model. This level of effort is necessary to serve the projected 150 participants across three program sites."

# SHEET 4: NARRATIVE TEMPLATES

> Frameworks for the most common sections in grant proposals.

---

## Section 1: Organizational Background (1 page)

**Template structure:**

| Paragraph | Content | Word Count Guide |
|:---------:|---------|:----------------:|
| 1 | Mission, founding year, legal status | 50-75 words |
| 2 | Programs and services overview | 75-100 words |
| 3 | Population served and geographic scope | 50-75 words |
| 4 | Key accomplishments and track record | 75-100 words |
| 5 | Annual budget, staff size, board overview | 50-75 words |

---

## Section 2: Statement of Need (1-2 pages)

**Template structure:**

| Paragraph | Content | Tips |
|:---------:|---------|------|
| 1 | Open with a compelling data point or story | Hook the reader |
| 2 | Define the problem (local data, scope, trends) | Use local data, not just national |
| 3 | Who is affected? (demographics, disparities) | Be specific about your community |
| 4 | Root causes and contributing factors | Show you understand the why |
| 5 | Consequences of inaction | What happens if nothing changes? |
| 6 | Gap in current services | Why existing resources are insufficient |
| 7 | Bridge to your solution | "Therefore, [Org Name] proposes..." |

**Data sources to cite:**

- Census data (census.gov)
- County health rankings
- State education data
- Community needs assessments
- Your own program data from prior years

---

## Section 3: Project Description / Approach (2-3 pages)

**Template structure:**

| Section | Content |
|---------|---------|
| Overview | 2-3 sentence program summary |
| Goals | 2-3 broad goals the project will achieve |
| Objectives | 3-5 SMART objectives (Specific, Measurable, Achievable, Relevant, Time-bound) |
| Activities | Detailed description of what you will do |
| Timeline | Major milestones by quarter |
| Staffing | Who will do what |
| Partnerships | Collaborating organizations and their roles |
| Evidence Base | Research supporting your approach |

**SMART Objective examples:**

- "By December 2027, 80% of program participants will demonstrate improved [outcome] as measured by [tool]."
- "Within 12 months, the program will serve 150 youth ages 14-18 across 3 community sites."
- "By project end, 70% of participants will report increased [behavior/knowledge] on pre/post assessments."

# FORMULA REFERENCE GUIDE

---

## Pipeline Tracking

**Win rate:**
```
=AwardedGrants / (AwardedGrants + DeclinedGrants) * 100
```

**Average grant size:**
```
=TotalAmountAwarded / NumberOfAwards
```

**Revenue per application:**
```
=TotalFunding / TotalApplicationsSubmitted
```

## Budget Calculations

**Personnel cost:**
```
=AnnualSalary * PercentTimeOnProject * (MonthsOnProject / 12)
```

**Fringe benefits:**
```
=SalaryOnProject * FringeRate
```

**Indirect costs:**
```
=TotalDirectCosts * NegotiatedRate
```

**Grant as % of total budget:**
```
=GrantRequest / TotalProjectBudget * 100
```

## Outcomes

**% of target achieved:**
```
=ActualResult / Target * 100
```

**Output per dollar:**
```
=TotalOutputs / GrantAmount
```

## Conditional Formatting

**Deadline within 14 days (red):**
```
=AND(Deadline - TODAY() <= 14, Deadline - TODAY() >= 0)
```

**Deadline within 30 days (yellow):**
```
=AND(Deadline - TODAY() <= 30, Deadline - TODAY() > 14)
```

**Compatibility score 4+ (green):**
```
=WeightedScore >= 4
```

**Grant awarded (green row):**
```
=Status = "Awarded"
```

**Grant declined (gray row):**
```
=Status = "Declined"
```

**Outcome below 80% of target (orange):**
```
=PercentAchieved < 80
```

---

> **NOTE:** Grant writing success is built on relationships, not just applications. The organizations with the highest win rates are those that build relationships with program officers before submitting proposals. Attend funder webinars, request pre-submission conversations, and steward existing funders well — renewals are easier than new awards. This toolkit gives you the system; your passion and program quality are what win the funding. Allow 6-12 months from first prospect research to first award for new funder relationships.
