# Rental Property Manager

Welcome to your Rental Property Manager. This Notion template gives you everything you need to manage your rental portfolio: property tracking, tenant management, lease monitoring, rent collection, maintenance logging, expense/income tracking, vacancy analysis, and tax-ready Schedule E summaries.

---

## Quick-Start Guide (30 Minutes)

1. **Duplicate this template** to your Notion workspace.
2. **Add your properties** to the Property Portfolio Dashboard (Section 1). Start with the basics: address, unit count, current rent.
3. **Add your tenants** to the Tenant Database (Section 2). Link them to their property.
4. **Enter current lease information** in the Lease Tracker (Section 3).
5. **Start logging rent payments** as they come in (Section 5).
6. **Add any open maintenance requests** (Section 4).
7. **Begin logging income and expenses** as they occur (Section 6).

The dashboard, vacancy monitor, and tax summary will auto-populate as you enter data.

### Move-In / Move-Out Checklist

Use this for every tenant transition:

**Move-In:**
- [ ] Lease signed and filed
- [ ] Security deposit collected and deposited in escrow/trust account
- [ ] First month's rent collected
- [ ] Move-in inspection completed (photos taken, checklist signed by tenant)
- [ ] Keys, fobs, garage remotes issued (document quantity)
- [ ] Utility transfer confirmed (or utility billing set up)
- [ ] Welcome packet provided (emergency contacts, trash schedule, rules, parking info)
- [ ] Renter's insurance verified (if required)

**Move-Out:**
- [ ] Written notice received (date: ___)
- [ ] Move-out inspection scheduled
- [ ] Move-out inspection completed (photos taken, checklist signed)
- [ ] Damages assessed and documented
- [ ] Security deposit disposition letter sent (within state-required timeframe)
- [ ] Security deposit returned (minus deductions, if any)
- [ ] Keys, fobs, remotes returned
- [ ] Utilities transferred back to owner
- [ ] Unit cleaned and turnover repairs completed
- [ ] Unit listed for new tenant

### Renewal Workflow

When a lease enters the 90-day window:

1. **90 days out:** Review current rent vs. market rent. Decide on renewal terms.
2. **75 days out:** Send renewal offer letter to tenant with proposed terms and rent adjustment.
3. **60 days out:** Follow up if no response. Begin preparing for potential vacancy (listing, showing schedule).
4. **45 days out:** Deadline for tenant decision. If no renewal, begin marketing the unit.
5. **30 days out:** If renewing, send new lease for signature. If vacating, finalize move-out timeline.
6. **Move-out day:** Conduct move-out inspection and begin turnover.

### Lease Type Definitions

| Type | Description |
|---|---|
| Fixed-term | Set start and end date (e.g., 12 months). Must be renewed or it expires. |
| Month-to-month | Continues indefinitely with 30-day notice to terminate by either party. |
| Fixed-to-MTM | Fixed term that automatically converts to month-to-month after expiration. |

### Commonly Missed Deductions Checklist

Review this list annually to make sure you're capturing everything:

- [ ] **Home office** — If you manage your properties from a dedicated home office space
- [ ] **Mileage** — Driving to/from properties for management, maintenance, showings (keep a log)
- [ ] **Phone & internet** — Percentage used for property management communications
- [ ] **Software & subscriptions** — Property management tools, accounting software, listing sites
- [ ] **Education** — Books, courses, seminars related to real estate investing
- [ ] **Bank fees** — Fees on accounts used for rental income
- [ ] **Closing costs** — Certain closing costs from purchase (points, title insurance)
- [ ] **Depreciation** — Make sure you're depreciating the building (not land) over 27.5 years
- [ ] **Improvements** — Capital improvements must be depreciated, not expensed (but don't miss them)
- [ ] **Travel** — If you travel to inspect or manage out-of-area properties
- [ ] **Pest control** — Routine and reactive pest control services
- [ ] **Landscaping & snow removal** — If landlord-paid
- [ ] **Startup costs** — Costs incurred before your first tenant (within limits)

### Tax Prep Checklist

Before meeting with your accountant or filing:

- [ ] All rental income entered in income log
- [ ] All expenses entered and categorized in expense log
- [ ] Receipts linked/uploaded for all expenses over $75
- [ ] Depreciation calculated (or confirmed with accountant)
- [ ] Security deposit transactions documented (not income until forfeited)
- [ ] 1099s received from property managers or other payers
- [ ] Mileage log completed
- [ ] Schedule E summary generated from this template
- [ ] Prior year Schedule E available for comparison

- [ ] Log any new rent payments received
- [ ] Update status on open maintenance requests
- [ ] Check for new tenant communications and log them
- [ ] Log any expenses incurred this week

## Monthly Routine (15-20 minutes)

- [ ] Complete the monthly rent roll (mark all units as paid/unpaid/late)
- [ ] Review lease expiration alerts
- [ ] Update vacancy status
- [ ] Review and categorize all expenses for the month
- [ ] Check per-property P&L

## Quarterly Routine (30 minutes)

- [ ] Review portfolio dashboard metrics
- [ ] Assess vacancy rate trends
- [ ] Review maintenance cost trends per property
- [ ] Evaluate rent vs. market rates (adjust if leases are renewing)
- [ ] Update property values (estimated)

## Annual Routine (60 minutes)

- [ ] Generate Schedule E tax summary
- [ ] Complete tax prep checklist
- [ ] Review insurance coverage and premiums
- [ ] Update property detail cards (value, mortgage balance)
- [ ] Evaluate portfolio performance (which properties to keep, sell, or improve)
- [ ] Review and update vendor database
