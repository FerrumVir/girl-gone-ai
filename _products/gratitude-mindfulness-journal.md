# Gratitude & Mindfulness Journal — Daily Practice Template

> Make a copy of this document to start using it. A structured 90-day mindfulness journal with 5-minute daily prompts, weekly intention setting, monthly reflection, meditation tracking, mood patterns, and a library of 100+ positive affirmations.

---

> **SETUP GUIDE — Start in 5 Minutes**
>
> 1. Create a new Google Doc or Sheets document (or print this template)
> 2. Begin with the Daily Gratitude Practice each morning (5 minutes)
> 3. Complete the Evening Mindfulness Check-in before sleep (3 minutes)
> 4. Use the Weekly Intention page every Sunday or Monday
> 5. Complete the Monthly Reflection on the last day of each month
> 6. Track your meditation practice on the Meditation Log
>
> **Tip:** Do not try to be profound. The most powerful entries are honest and specific: "I'm grateful for the way my coffee tasted this morning" is better than "I'm grateful for life." Specificity is the key to feeling gratitude, not just writing about it.

# TRACKING FORMULAS (for Google Sheets version)

---

## Gratitude Tracking

**Completion rate:**
```
=COUNTIF(Completed, "Y") / DayNumber * 100
```

**Average mood:**
```
=AVERAGE(MoodColumn)
```

**Meditation streak:**
```
=Count consecutive "Y" entries from today backward
```

## Mood Pattern Analysis

**Average mood when exercised:**
```
=AVERAGEIF(ExerciseColumn, "Y", MoodColumn)
```

**Mood trend (week over week):**
```
=AVERAGE(ThisWeekMoods) - AVERAGE(LastWeekMoods)
```

## Conditional Formatting

**Mood 8+ (green):**
```
=MoodCell >= 8
```

**Mood 4 or below (red/orange):**
```
=MoodCell <= 4
```

**Meditation completed (green):**
```
=MeditatedCell = "Y"
```

**Streak broken (gray):**
```
=MeditatedCell = "N"
```

> **NOTE:** This journal is undated — start any day. The 90-day framework gives you three full months to build a lasting habit. Research shows it takes 66 days on average to form a habit, so 90 days gives you a comfortable buffer. Do not judge your entries. Do not compare Day 1 to Day 90. Simply show up, write honestly, and trust the process. The transformation happens in the accumulation of small, consistent acts of awareness.
