# Course Creator Launch Kit — Google Sheets Template

> Make a copy of this spreadsheet to start using it. Plan, price, and launch your online course with a complete system covering curriculum design, pricing strategy, launch timeline, email sequences, webinar planning, revenue projections, and student tracking.

---

> **SETUP GUIDE — Get Running in 20 Minutes**
>
> 1. Create a new Google Sheets document
> 2. Create 7 tabs/sheets and name them: Curriculum Outline, Pricing Strategy, Launch Timeline, Email Sequences, Webinar Planner, Revenue Projections, Student Tracker
> 3. Copy each section below into its corresponding sheet
> 4. Enter all formulas as documented (marked with `FORMULA:`)
> 5. Start with Curriculum Outline — map your course content first, then price it
> 6. Work backward from your launch date to build the timeline
>
> **Tip:** Your first course does not need to be perfect. It needs to exist. Use this kit to get from idea to enrolled students in 8-12 weeks. You can improve V2 after you have paying students and real feedback.

# FORMULA REFERENCE GUIDE

---

## Revenue Calculations

**Net revenue:**
```
=GrossRevenue * (1 - RefundRate) * (1 - PlatformFee)
```

**Break-even students:**
```
=TotalCosts / PricePerStudent
```

**Conversion rate:**
```
=Students / TotalLeads * 100
```

**Revenue per lead:**
```
=TotalRevenue / TotalLeads
```

## Content Progress

**Course completion %:**
```
=LessonsCompleted / TotalLessons * 100
```

**Content production %:**
```
=RecordedLessons / PlannedLessons * 100
```

## Conditional Formatting

**Task completed (green):**
```
=Status = "Complete"
```

**Task overdue (red):**
```
=AND(Status <> "Complete", Deadline < TODAY())
```

**Module completion below 50% (orange):**
```
=CompletionRate < 0.5
```

**Revenue target met (green):**
```
=ActualRevenue >= TargetRevenue
```
