## SECTION 1: ERGONOMIC SETUP CHECKLIST

### The 15-Point Ergonomic Assessment

Go through each item and adjust your setup until all boxes are checked.

```
MONITOR POSITION:
- [ ] Top of screen at or slightly below eye level
- [ ] Monitor distance: arm's length away (20-26 inches)
- [ ] Screen tilted 10-20 degrees backward
- [ ] No glare from windows or lights on screen
- [ ] If using laptop only: elevated on a stand + external keyboard

CHAIR:
- [ ] Feet flat on floor (or on footrest)
- [ ] Knees at 90-degree angle (thighs parallel to floor)
- [ ] Lumbar support in the curve of lower back
- [ ] Seat pan doesn't press into back of knees (2-3 finger gap)
- [ ] Armrests at height where shoulders are relaxed (not hunched)

DESK & KEYBOARD:
- [ ] Elbows at 90-degree angle when typing
- [ ] Wrists neutral (not bent up or down) — wrist rest optional
- [ ] Mouse at same height as keyboard
- [ ] Frequently used items within arm's reach (no repeated stretching)

GENERAL:
- [ ] 20-20-20 rule reminder set (every 20 min, look 20 feet away for 20 seconds)
- [ ] Stand/move at least 5 minutes every hour
```

### Common Ergonomic Problems & Fixes
```
PROBLEM: Neck pain/stiffness
FIX: Monitor too low. Raise it. If using a laptop, get a stand + external keyboard.

PROBLEM: Lower back pain
FIX: Chair lacks lumbar support. Add a lumbar pillow or roll a towel.

PROBLEM: Wrist pain/tingling
FIX: Wrists bent while typing. Lower keyboard or raise chair. Keyboard tray helps.

PROBLEM: Shoulder tension
FIX: Armrests too high (shoulders shrugging) or too low (leaning to one side). Adjust armrest height until shoulders are relaxed.

PROBLEM: Eye strain/headaches
FIX: Screen too bright, too close, or has glare. Adjust brightness to match room. Use blue-light filter after 7pm. Follow 20-20-20 rule.

PROBLEM: Hip tightness
FIX: Sitting too long. Set a timer to stand every 45-60 minutes. Consider a standing desk converter for alternating.
```

## SECTION 3: PRODUCTIVITY ZONE DESIGN

### Room Layout Templates

**Layout A: Dedicated Room (Spare Bedroom)**
```
┌─────────────────────────────────────────────┐
│ WINDOW                                       │
│ ┌──────────┐                                │
│ │  DESK    │ ← Face window or perpendicular │
│ │(monitor) │   (not window behind you —     │
│ └──────────┘    causes glare on calls)      │
│                                             │
│ ┌────┐                                      │
│ │BOOK│ ← Reading/break zone                 │
│ │NOOK│   (chair + lamp + books)             │
│ └────┘                                      │
│                                             │
│ ┌──────────┐                                │
│ │ STORAGE  │ ← Shelving for supplies        │
│ └──────────┘                                │
│                                    [DOOR]   │
└─────────────────────────────────────────────┘

PRINCIPLES:
- Desk faces toward or perpendicular to window (not back to it)
- Reading/break zone separate from desk (mental separation)
- Door visible from desk (reduces startle interruptions)
- Storage organized and accessible
```

**Layout B: Corner of Living Room/Bedroom**
```
┌─────────────────────────────────────────────┐
│                                             │
│                    LIVING SPACE              │
│                                             │
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ │ ← Visual divider
│ ┌──────────┐                                │  (bookshelf, plants,
│ │  DESK    │ ← Faces wall or corner         │   curtain, screen)
│ │(compact) │                                │
│ └──────────┘                                │
│ [PLANT] [LAMP]                              │
└─────────────────────────────────────────────┘

PRINCIPLES:
- Create visual boundary between "work" and "life" zones
- Use a room divider, tall bookshelf, curtain, or even a large plant
- Face AWAY from the living space (reduces distractions)
- End-of-day ritual: close laptop, turn off desk lamp = "leaving work"
```

**Layout C: Closet Office (Cloffice)**
```
┌────────────┐
│ ┌────────┐ │
│ │ SHELF  │ │ ← Floating shelves above desk
│ ├────────┤ │
│ │  DESK  │ │ ← Wall-mounted or narrow desk
│ │(24"deep)│ │    (IKEA LAGKAPTEN 47" fits most closets)
│ ├────────┤ │
│ │ CHAIR  │ │ ← Task chair (compact)
│ └────────┘ │
│            │
└────────────┘
  ← Closet doors (close when done = work disappears)

PRINCIPLES:
- Remove closet doors or use bifold doors
- Wall-mounted desk saves floor space
- Floating shelves above for storage
- LED strip light or puck lights for overhead
- Close doors at end of day = perfect work/life boundary
```

### Video Call Background Setup
```
GOOD BACKGROUNDS:
- Bookshelf (not too cluttered — curated)
- Clean wall with one piece of art
- Plants (creates depth and warmth)
- Tidy, minimal space behind you

BAD BACKGROUNDS:
- Bed visible (unprofessional perception)
- Harsh overhead lighting (casts shadows under eyes)
- Window behind you (silhouettes your face)
- Clutter, laundry, mess

QUICK FIXES:
- Hang a curtain or tapestry behind your call area
- Add 2-3 plants at different heights
- Ensure light source is IN FRONT of you (not behind)
- Declutter only the area the camera sees (everything else can be messy)
```

## SECTION 4: DAILY ROUTINE TEMPLATES

### Template A: The Standard 9-5 Remote Routine
```
6:30 AM — Wake up (no phone for 30 min)
6:30-7:00 — Morning routine (shower, coffee, breakfast)
7:00-7:30 — Movement (walk, stretch, gym, yoga)
7:30-8:00 — Personal time (read, journal, plan day)

8:00-8:15 — Workday start: review calendar, check messages
8:15-10:00 — DEEP WORK BLOCK 1 (hardest task, no meetings)
10:00-10:15 — Break: walk, stretch, snack
10:15-12:00 — DEEP WORK BLOCK 2 or collaborative work

12:00-1:00 — LUNCH (away from desk, no screens)

1:00-1:15 — Quick inbox check, respond to urgent items
1:15-3:00 — Meetings / collaborative work / admin
3:00-3:15 — Break: movement, fresh air
3:15-5:00 — Focused work: wrap up tasks, plan tomorrow
5:00-5:15 — END OF DAY RITUAL: close tabs, update task list, write tomorrow's top 3

5:15 PM — Workday ends. Walk away from desk.

EVENING: phone-free dinner, hobbies, rest, early wind-down
```

### Template B: The Early Bird (Creative Professional)
```
5:00 AM — Wake up
5:00-5:30 — Coffee + journaling (no phone)
5:30-8:00 — CREATIVE DEEP WORK (writing, design, strategy — before anyone else is awake)
8:00-8:30 — Breakfast + movement

8:30-9:00 — Check messages, respond, plan day
9:00-12:00 — Meetings + collaborative work (scheduled in this window)
12:00-12:30 — Lunch (away from desk)
12:30-2:00 — Admin tasks, emails, shallow work
2:00 PM — Workday ends (the tradeoff for starting at 5am)

AFTERNOON/EVENING: family, hobbies, exercise, rest
```

### Template C: The Flexible Schedule (Async Worker)
```
7:00 AM — Wake up, morning routine
7:30-8:00 — Walk or exercise (phone-free)
8:00-8:30 — Plan day: what are today's 3 most important tasks?

BLOCK 1: 8:30-11:00 — Deep work (MIT #1 and #2)
BREAK: 11:00-11:30 — Movement, snack

BLOCK 2: 11:30-1:00 — Communication window (Slack, email, meetings)
BREAK: 1:00-2:00 — Long lunch / errand / nap / walk

BLOCK 3: 2:00-4:00 — Deep work (MIT #3 + overflow)
BREAK: 4:00-4:15 — Movement

BLOCK 4: 4:15-5:30 — Wrap-up, planning, low-energy admin

5:30 PM — Done. End-of-day ritual.

KEY PRINCIPLE: Group communication. Protect deep work blocks. Don't let Slack interrupt focus time.
```

### The End-of-Day Shutdown Ritual (5 minutes)
```
Do this EVERY day to separate work from life:

1. Write tomorrow's top 3 priorities (so your brain can stop holding them)
2. Close all browser tabs and applications
3. Turn off your monitor
4. Stand up and push in your chair
5. Say out loud (or think clearly): "Workday complete"
6. Leave the workspace (walk to another room, even briefly)
7. Do NOT check work email/Slack until tomorrow's start time

WHY: Without commute, your brain has no "off switch." This creates one.
```

### Solutions by Distraction Type

**For Digital Distractions:**
```
- Phone in another room during focus blocks
- Website blocker (Cold Turkey, Freedom, SelfControl) during work
- Batch Slack/email to 3 check times (not continuous)
- All notifications off except calendar and calls
- Use "Focus Mode" on phone and computer
```

**For Environmental Distractions:**
```
NOISE:
- Noise-cancelling headphones (ANC on, even without music)
- White/brown noise (mynoise.net, Noisli app, or YouTube "brown noise 10hr")
- Close the door + "do not disturb" sign during focus blocks
- Communicate focus schedule to household members

VISUAL:
- Clear desk policy: only current task materials visible
- End of each day: reset desk to clean state
- Face away from high-traffic areas
- Minimize decorative items that catch your eye
```

**For Internal Distractions:**
```
UNCLEAR PRIORITIES:
- Write top 3 tasks before starting (the night before is best)
- Time-block your calendar (assign tasks to specific times)
- If stuck: "What's the ONE thing that if done, makes everything else easier?"

LOW ENERGY:
- Align hard tasks with energy peaks (morning for most people)
- Take real breaks (movement, not scrolling)
- Hydrate (keep water at desk, refill every hour)
- Afternoon slump: 10-min walk + cold water > coffee

LONELINESS:
- Schedule virtual co-working sessions (Focusmate.com)
- Body-doubling: video call with a friend, both work silently
- Weekly in-person meetup (cafe, library, coworking space)
- End-of-day check-in call with a friend/partner
```

### The Focus Block Protocol
```
SETUP (2 minutes before focus block):
- Close all apps except the ONE you need for this task
- Phone on silent, face-down, in another room (or drawer)
- Noise-cancelling headphones on (with brown noise or silence)
- Water bottle full and within reach
- One sentence: "For the next [45/60/90] minutes, I am doing: ________"
- Set timer

DURING (45-90 minutes):
- ONE task only
- If a thought/to-do interrupts: write it on a "parking lot" notepad (don't act on it)
- If urge to check phone/Slack: notice it, return to work (it passes in 60 seconds)
- No "quick" email/Slack checks (there's no such thing)

AFTER (5 minutes):
- Stand up, stretch, walk briefly
- Check parking lot: anything actually urgent? (rarely)
- Quick Slack/email scan (respond only to urgent)
- Reset for next block or take a longer break
```

## SECTION 7: WORK-FROM-HOME BOUNDARIES

### Boundary Templates (For Communicating with Household)

**Template: The Family/Partner Conversation**
```
"I need to set up some work boundaries so I can be more present when I'm NOT working. Here's what I'm thinking:

WORK HOURS: [TIME] to [TIME]
During this time:
- My office door closed = do not interrupt unless emergency
- I'll check in at [BREAK TIMES] for anything non-urgent
- Emergency definition: safety issue, urgent logistics, someone at the door

AFTER WORK:
- I'm 100% present from [TIME] onward
- I won't check Slack/email after [TIME]
- [DAY] is my fully phone-free evening

Does this work? What do you need from me to make this feel fair?"
```

**Template: The Roommate Agreement**
```
"Hey — now that we're both working from home, can we align on a few things?

NOISE AGREEMENT:
- Quiet hours: [MORNING BLOCK] and [AFTERNOON BLOCK]
- Headphones for calls (both of us)
- Loud tasks (vacuuming, music, calls in common areas) outside of [TIMES]

SPACE AGREEMENT:
- I work in [MY SPACE], you work in [YOUR SPACE]
- Common areas: whoever has a call gets priority for quiet
- Shared kitchen: stagger lunch to avoid crowding

SOCIAL AGREEMENT:
- Let's have lunch together on [DAY] (intentional social time)
- Otherwise, work hours = minimal chitchat (no offense, just focus)

Sound fair?"
```

**Template: The Manager/Team Communication**
```
"Hey [MANAGER/TEAM] — I want to share my availability approach so we're aligned:

MY WORK HOURS: [TIME] to [TIME] [TIMEZONE]
DEEP FOCUS BLOCKS: [TIMES — e.g., 9-11am and 2-4pm] — I batch messages outside these
RESPONSE TIME: Within [X] hours for Slack, within [X] hours for email
BEST FOR URGENT: [METHOD — call, text, specific Slack channel]
MEETINGS: I prefer [DAYS/TIMES] — my calendar reflects my availability

I've found this structure helps me deliver better work and be more responsive during my communication windows. Let me know if anything needs adjustment."
```
