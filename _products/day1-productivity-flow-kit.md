# SECTION 2: TASK BATCHING FRAMEWORK

---

## How Task Batching Works

Instead of switching between different types of tasks all day (which costs you 23 minutes of refocus time per switch), group similar tasks together.

## Task Categories

### Category 1: DEEP WORK (requires peak energy)

```
Definition: Tasks requiring sustained focus, creativity, or complex thinking.
Schedule during: Your peak hours only.
Environment: No notifications, no interruptions, door closed.
Minimum block size: 60 minutes.
```

**My Deep Work tasks this week:**
- [ ] [TASK]
- [ ] [TASK]
- [ ] [TASK]
- [ ] [TASK]

### Category 2: COMMUNICATION (moderate energy)

```
Definition: All emails, messages, calls, and meetings.
Schedule during: Moderate energy hours.
Batch strategy: Check email/messages 2-3 times per day, not constantly.
Time blocks: 30-minute communication blocks.
```

**My Communication blocks:**
- Morning batch: [TIME] — ___ minutes
- Midday batch: [TIME] — ___ minutes
- End-of-day batch: [TIME] — ___ minutes

### Category 3: ADMIN (low energy)

```
Definition: Filing, organizing, scheduling, updating, data entry.
Schedule during: Low energy hours (post-lunch, end of day).
Batch strategy: Save all admin for one block rather than sprinkling throughout day.
```

## Focus Session Template

> Use this for every deep work block. Duration depends on your energy and task.

### Pre-Session Setup (2 minutes)

- [ ] Clear workspace of distractions
- [ ] Close all irrelevant tabs/apps
- [ ] Phone: silenced and face-down (or in another room)
- [ ] Notifications: OFF (Do Not Disturb mode)
- [ ] Water and snack within reach
- [ ] Timer set for: ___ minutes

### Session Definition

```
TASK: [SPECIFIC TASK — not vague, not multiple tasks]
SESSION LENGTH: [25 / 45 / 60 / 90] minutes
SUCCESS CRITERIA: What does "done" look like for this session?
[DESCRIBE — e.g., "First draft of email sequence complete" or "3 pages written"]
```

### During the Session

**If distracted by a thought/idea:** Write it in the Parking Lot (below) and return to task.
**If stuck:** Switch to the easiest part of the task, or set a 5-min timer and push through.
**If timer goes off and you're in flow:** Optional — extend by 15 minutes max, then break.
