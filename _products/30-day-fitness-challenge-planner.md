# 30-Day Fitness Challenge Planner — Google Sheets Template

> Make a copy of this spreadsheet to start using it. A complete 30-day progressive workout program with daily schedules, exercise library, progress tracking, meal plan integration, and weekly check-ins. Three levels: Beginner, Intermediate, Advanced.

---

> **SETUP GUIDE — Get Running in 10 Minutes**
>
> 1. Create a new Google Sheets document
> 2. Create 6 tabs/sheets and name them: 30-Day Schedule, Exercise Library, Progress Tracker, Meal Plan, Rest Day Protocols, Weekly Check-In
> 3. Copy each section below into its corresponding sheet
> 4. Choose your level: Beginner, Intermediate, or Advanced
> 5. Take Day 1 measurements and photos before starting
> 6. Follow the daily schedule — check off each workout as you complete it
>
> **Tip:** The best workout program is one you actually finish. Choose the level that feels "moderate" — not too easy, not intimidating. You can always level up for your next 30 days.

## Intermediate Adjustments

| Change from Beginner | Details |
|---------------------|---------|
| Add 1 set to each exercise | 3 sets instead of 2 |
| Increase cardio duration | +10 minutes per session |
| Add weight/resistance | Dumbbells or bands for all exercises |
| Reduce rest between sets | 45 sec instead of 60 sec |
| Add plyometric variations | Jump squats, burpees, etc. |

---

## Advanced Adjustments

| Change from Intermediate | Details |
|--------------------------|---------|
| Add 1 more set (4 total) | Higher volume |
| Superset exercises | Pair exercises back-to-back |
| Add explosive movements | Box jumps, clap push-ups |
| Reduce rest to 30 seconds | Higher intensity |
| Add AM/PM split option | Two shorter sessions per day |

# FORMULA REFERENCE GUIDE

---

## Progress Calculations

**Completion rate:**
```
=COUNTIF(CompletedColumn, "Y") / 30 * 100
```

**Measurement change:**
```
=Day30Value - Day1Value
```

**Percentage improvement:**
```
=(Day30 - Day1) / Day1 * 100
```

## Nutrition

**TDEE estimate:**
```
=BodyWeight * ActivityMultiplier (12/14/16)
```

**Protein target (grams):**
```
=BodyWeight * ProteinMultiplier (0.8/1.0/1.2)
```

**Calorie deficit/surplus:**
```
=TDEE +/- CalorieAdjustment
```

## Conditional Formatting

**Workout completed (green):**
```
=CompletedCell = "Y"
```

**Workout missed (red):**
```
=AND(CompletedCell = "N", DayNumber <= TODAY() - StartDate)
```

**Energy level low (orange, <= 2):**
```
=EnergyCell <= 2
```

**Improvement positive (green):**
```
=ChangeValue > 0
```

**Improvement negative (red):**
```
=ChangeValue < 0
```

> **NOTE:** This program is designed for generally healthy adults. Consult a physician before starting any exercise program, especially if you have pre-existing conditions, injuries, or have been sedentary for an extended period. Listen to your body — sharp pain is never okay. Muscle soreness 24-48 hours after exercise is normal; joint pain or sharp sensations during exercise are not. Modify any exercise that causes discomfort and skip exercises that aggravate injuries.
