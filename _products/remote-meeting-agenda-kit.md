## 1:1 Meeting Templates

**Purpose:** Build relationship, remove blockers, discuss growth, align on priorities.

---

**BEFORE THE MEETING (Both parties):**
- [ ] Review action items from last 1:1

## Daily Standup Template

### Async Standup (Posted in Slack/Teams, No Meeting Needed)

**When:** Every workday by [TIME]
**Where:** [#channel-name]
**Duration:** 2 minutes to write, 5 minutes to read others'

---

```
## Daily Standup — [Your Name] — [Date]

**Yesterday:**
- [What I completed]
- [What I worked on]

**Today:**
- [What I'm focusing on]
- [Priority task]

**Blockers:**
- [Anything preventing progress — or "None"]

**FYI:**
- [Optional: heads up, question, or context for the team]
```

---

### Synchronous Standup (15 minutes max)

**Rules:**
- Start on time, end on time (set a timer)
- Each person gets 2 minutes maximum
- Not a problem-solving session (take it offline)
- Camera on (optional but encouraged for connection)
- Stand up if you can (keeps it short)

---

**Facilitator Script:**

```
"Good morning everyone. Quick standup — 2 minutes each.
Remember: what you did, what you're doing, any blockers.
Save deep discussions for after. Let's go.
[Name], you're up first."

[After everyone has spoken:]

"Any blockers that need immediate attention?
[If yes:] [Name] and [Name], can you connect after this to sort that out?
Alright, go have a great day. Same time tomorrow."
```

### Alternative Retro Formats

**Format 1: Start / Stop / Continue**
- START doing: [things we should begin]
- STOP doing: [things that aren't working]
- CONTINUE doing: [things that are working]

**Format 2: 4Ls**
- LOVED: [what was great]
- LEARNED: [what we discovered]
- LACKED: [what was missing]
- LONGED FOR: [what we wish we had]

**Format 3: Sailboat**
- Wind (what pushed us forward):
- Anchor (what held us back):
- Rocks (risks we navigated):
- Island (our goal — did we reach it?):

**Format 4: One Word**
Each person shares ONE WORD describing the sprint. Then discuss themes that emerge.

## Brainstorming Session Template

### Brainstorm Meeting (45-60 minutes)

**Purpose:** Generate ideas. Quantity over quality. No judging.

---

```
## Brainstorming Session
Date: ___________
Topic/Challenge: ___________________________________
Facilitator: ___________
Participants: ___________

### Ground Rules (Read aloud at start)
1. No bad ideas (seriously — the weird ones often lead to the best ones)
2. Build on others' ideas ("Yes, and...")
3. Quantity over quality (we'll filter later)
4. No criticism during ideation (separate generating from evaluating)
5. One conversation at a time
6. Write everything down

### Warm-Up (5 min)
[Quick creative exercise to get brains in generative mode]
Options:
- "In 60 seconds, write down as many uses for a paperclip as you can"
- "Worst possible solution to our problem" (then flip it)
- "How would [FAMOUS PERSON: Elon Musk/Oprah/a 5-year-old] solve this?"

### Problem Statement (5 min)
"How might we [REFRAME THE CHALLENGE as a question]?"

Example: "How might we reduce customer churn in the first 30 days?"

### Ideation Round 1: Silent Brainstorm (10 min)
Everyone writes ideas individually (sticky notes, Miro, Google Doc).
No talking. No reacting. Just write.
Target: 10+ ideas per person.

### Share & Cluster (10 min)
Each person shares their ideas (30 seconds each).
Facilitator groups similar ideas into themes/clusters.
Name each cluster.

### Ideation Round 2: Build & Combine (10 min)
Look at the clusters. Now build on each other's ideas:
- "What if we combined [IDEA A] with [IDEA B]?"
- "What's a more extreme version of [IDEA]?"
- "What's the simplest version of [IDEA]?"

### Prioritize / Vote (10 min)
Each person gets 3 votes (dots/stars). Vote for the ideas you think
have the most potential. Can put multiple votes on one idea.

**Top voted ideas:**
1. [IDEA] — [X] votes
2. [IDEA] — [X] votes
3. [IDEA] — [X] votes

### Next Steps (5 min)
For the top 3 ideas:
| Idea | Next Step | Owner | Deadline |
|------|-----------|-------|----------|
| | | | |
| | | | |
| | | | |
```

## Meeting Best Practices

### Before the Meeting

- [ ] Does this meeting need to happen? (Could it be an email, async update, or doc?)
- [ ] Who actually needs to be there? (Minimum viable attendees)
- [ ] Is there a clear agenda shared 24+ hours in advance?
- [ ] Is pre-work clearly defined? (Read this doc, form an opinion, come prepared)
- [ ] Is the meeting length appropriate? (Default to 25 or 50 minutes, not 30/60)
- [ ] Is a facilitator designated?
- [ ] Is a note-taker designated?

### During the Meeting

- [ ] Start on time (don't wait for latecomers — it punishes the punctual)
- [ ] State the purpose in the first 30 seconds ("We're here to [DECIDE/DISCUSS/ALIGN ON]...")
- [ ] Follow the agenda (table off-topic items for later)
- [ ] Watch the time (facilitator's job)
- [ ] Ensure everyone speaks (directly invite quiet people: "[Name], what's your take?")
- [ ] Capture action items IN REAL TIME (don't try to remember later)
- [ ] End with: "What did we decide? Who's doing what by when?"
- [ ] End on time or early (never late)

### After the Meeting

- [ ] Send notes/action items within 2 hours (while memory is fresh)
- [ ] Format: Decision made + Action items (owner + due date) + Key discussion points
- [ ] Follow up on action items at next meeting or async check-in
- [ ] If a recurring meeting consistently has no agenda or low attendance — cancel it

### Meeting Hygiene Rules

| Rule | Why |
|------|-----|
| No meeting without an agenda | Prevents aimless conversations |
| Default to 25 minutes (not 30) | Forces focus, gives buffer between meetings |
| Camera on for meetings under 8 people | Builds connection, prevents multitasking |
| One person talks at a time | Especially important on video calls |
| If you're not contributing or learning, leave | People's time is valuable |
| Friday afternoons = no meetings | Protect deep work time |
| Async first, sync second | Only meet when async won't work |
