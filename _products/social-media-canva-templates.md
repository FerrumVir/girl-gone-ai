## SECTION 1: BRAND BOARD TEMPLATE

### What Goes on Your Brand Board
Your brand board is your visual "source of truth." Reference it every time you create content.

```
BRAND BOARD COMPONENTS:
+--------------------------------------------------+
|  [YOUR BRAND NAME / LOGO]                        |
+--------------------------------------------------+
|  COLOR PALETTE                                   |
|  [PRIMARY] [SECONDARY] [ACCENT] [NEUTRAL] [BG]  |
|  #hex      #hex        #hex     #hex      #hex   |
+--------------------------------------------------+
|  TYPOGRAPHY                                      |
|  Headlines: [FONT NAME]                          |
|  Subheadlines: [FONT NAME]                       |
|  Body text: [FONT NAME]                          |
+--------------------------------------------------+
|  MOOD / AESTHETIC                                 |
|  [3-5 reference images showing your vibe]        |
+--------------------------------------------------+
|  BRAND VOICE                                     |
|  3 words: [WORD 1] [WORD 2] [WORD 3]           |
+--------------------------------------------------+
|  PHOTO STYLE                                     |
|  [Filter / editing style / lighting notes]       |
+--------------------------------------------------+
```

### Brand Board Creation Steps
```
Step 1: Open Canva → Custom size → 1920x1080px (or use "Brand Board" template)
Step 2: Divide into sections using lines or boxes
Step 3: Add your logo (or just brand name in your primary font)
Step 4: Add color swatches (5 circles/squares with hex codes below)
Step 5: Type your font names in each font style
Step 6: Add 3-5 inspiration images that match your aesthetic
Step 7: Add 3 brand voice words
Step 8: Save and reference for ALL future design work
```

## SECTION 2: COLOR PALETTE GUIDE

### The 5-Color Brand Palette System

**Color 1: Primary (40% usage)**
```
Your main brand color. Used for: headlines, CTAs, borders, accent elements.
Choose: A color that represents your brand personality.
- Bold/Energetic → Red, Orange, Hot Pink
- Calm/Professional → Navy, Dark Green, Deep Purple
- Friendly/Warm → Coral, Terracotta, Mustard
- Clean/Minimal → Black, Charcoal, Deep Teal
```

**Color 2: Secondary (30% usage)**
```
Complements your primary. Used for: subheadlines, icons, secondary buttons.
Choose: A color that pairs well without competing.
Tips: Use a color wheel — go analogous (next to primary) for harmony, or complementary (opposite) for contrast.
```

**Color 3: Accent (10% usage)**
```
The "pop" color. Used sparingly for: highlights, arrows, special callouts.
Choose: Something eye-catching that draws attention to key elements.
This color makes things stand out — use it intentionally.
```

**Color 4: Neutral (10% usage)**
```
Your text/body color. Used for: body text, borders, subtle elements.
Options: Charcoal (#333333), Dark Gray (#4A4A4A), Soft Black (#1A1A1A)
Avoid: Pure black (#000000) on white — it's harsh. Charcoal is softer.
```

**Color 5: Background (10% usage — but covers large areas)**
```
Your base canvas color. Used for: backgrounds, card fills.
Options: Pure White (#FFFFFF), Off-White (#F8F8F8), Cream (#FFF9F0), Light Gray (#F5F5F5)
Or: dark backgrounds for moody brands (#1A1A2E, #0D0D0D)
```

## SECTION 3: FONT PAIRING RECOMMENDATIONS

### The 3-Font System
```
Font 1 — HEADLINES: Bold, attention-grabbing, personality-driven
Font 2 — SUBHEADLINES: Pairs with headline, slightly lighter weight
Font 3 — BODY TEXT: Highly readable, clean, simple

Rule: Never use more than 3 fonts across ALL your content. Consistency = professionalism.
```

## SECTION 6: DESIGN RULES & BEST PRACTICES

### The 10 Design Rules for Non-Designers
```
1. ONE focal point per post. Don't make the viewer's eye bounce between competing elements.

2. White space is not wasted space. It makes content breathable and scannable.

3. Align everything. Left-aligned text, centered text — pick one and stick to it. No random floating elements.

4. Contrast is king. If text is on a photo, add a semi-transparent overlay so text is readable.

5. Consistent margins. Keep the same padding on all sides of every post (40-60px in Canva).

6. Limit colors. 2-3 colors per post from your brand palette. Not a rainbow.

7. Hierarchy matters. The most important thing (headline) should be the BIGGEST and first thing seen.

8. Grid view matters. Before posting, zoom out and look at how it fits with your last 6-9 posts.

9. Less text = more engagement. For posts: one idea. For carousels: one idea per slide.

10. Mobile first. Design on desktop, but always preview on mobile before publishing.
```

### Canva Quick Tips
```
- Use "Brand Kit" (even on free) to save your colors and fonts
- Duplicate designs (don't start from scratch each time)
- Use "Resize" feature to adapt one design to multiple platforms
- Lock background elements so you don't accidentally move them
- Use grids and frames for clean photo placement
- CMD/CTRL + D = duplicate element (fastest way to keep spacing consistent)
- Export at "highest quality" for Instagram (PNG for graphics, JPG for photos)
```

### Accessibility Checklist
```
- [ ] Text contrast ratio is sufficient (use WebAIM contrast checker)
- [ ] Don't rely on color alone to convey meaning
- [ ] Alt text added when uploading to platforms
- [ ] Text size readable on mobile (minimum 24pt for IG)
- [ ] Avoid thin fonts on busy backgrounds
- [ ] Captions included on video content
```

## SECTION 7: CONTENT BATCHING WORKFLOW

### Weekly Batch Creation Process
```
TIME BLOCK: 2-3 hours, one day per week

STEP 1 (20 min): Plan
- Review content calendar for the week
- List all posts needed (with platform and format)
- Gather any photos, screenshots, or assets needed

STEP 2 (60 min): Create
- Open Canva, duplicate your templates
- Create all static posts first (fastest)
- Then create carousel slides
- Then Story templates
- Use consistent elements across all posts

STEP 3 (20 min): Review
- Preview each at mobile size
- Check for typos
- Ensure brand consistency
- Verify dimensions are correct for each platform

STEP 4 (15 min): Export & Schedule
- Download all files (organized by platform and date)
- Upload to scheduler (Later, Planoly, Buffer, native schedulers)
- Add captions, hashtags, alt text
- Schedule for optimal times

STEP 5 (5 min): Filing
- Save master Canva files in organized folders
- Note what performed well (for next week's batch)
```
