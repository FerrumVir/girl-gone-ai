# ADHD Focus Timer & Task Manager
### A Complete Focus System Built for How Your Brain Actually Works

---

> **START HERE (30 seconds)**
>
> 1. Go to the Energy Check-In below
> 2. Pick your current energy level
> 3. Choose ONE task from the matching energy batch
> 4. Set your timer using the ADHD-Modified Pomodoro settings
> 5. Do the thing. Come back when the timer dings.
>
> That's it. Everything else is optional until you need it.

---

## How This System Works

Traditional Pomodoro: 25 min on, 5 min off, repeat. Fine for neurotypical brains.

**ADHD-Modified Pomodoro:** Flexible intervals based on your current energy and task type, built-in transition rituals so you don't lose 40 minutes between sessions, dopamine rewards baked into the structure, and permission to ride hyperfocus when it shows up.

# SECTION 3: TRANSITION RITUALS

---

> The ADHD tax on transitions: You finish one task, then lose 20-45 minutes scrolling, snacking, or staring before starting the next one. These rituals create a BRIDGE between tasks.

## The 3-Minute Transition Protocol

Do this EVERY TIME you finish a focus block and need to start a new task:

**Minute 1: CLOSE**

- [ ] Write one sentence: "I stopped here: [WHERE YOU LEFT OFF]"
- [ ] Save/close all files related to the previous task
- [ ] Physically push your chair back from the desk

**Minute 2: RESET**

- [ ] Stand up
- [ ] Take 3 deep breaths (in for 4, hold for 4, out for 6)
- [ ] Drink water
- [ ] Do ONE physical action: stretch, walk to the window, touch your toes

**Minute 3: OPEN**

- [ ] Say out loud: "I am now switching to [NEXT TASK]"
- [ ] Open ONLY what you need for the next task
- [ ] Set your timer
- [ ] Begin with the very first physical action

# SECTION 4: BODY DOUBLING SESSION TEMPLATES

---

## What Is Body Doubling?

Having another person present (physically or virtually) while you work. Their presence creates gentle external accountability that helps your brain engage. You don't need to talk, interact, or even do the same task.

---

## Solo Body Doubling Setup (Virtual)

**Before the session:**

- [ ] Choose a body doubling video/stream (YouTube "study with me," Focusmate, Discord study rooms)
- [ ] Set your environment (noise, lighting, workspace cleared)
- [ ] Write your task and timer length on a sticky note visible to you
- [ ] Put phone in another room or face-down

**Session template:**

```
Body Doubling Session — [DATE]

Platform/Video: [WHAT I'M USING]
Start Time: [TIME]
Planned Duration: [MINUTES]
My Task: [SPECIFIC TASK]
Timer Setting: [MINUTES]

--- Session Notes ---
How it went:
Did I stay on task? Y / N / Mostly
Distractions:
Would I use this setup again? Y / N
```

---

## Partner Body Doubling Template (for a friend/colleague)

Send this message to set up a session:

```
Hey! Want to body double with me? Here's how it works:

- We hop on a video call (cameras on, mics muted)
- We each say what we're working on
- We set a timer for [25/45/60] minutes
- We work silently
- Timer goes off, we check in briefly
- Repeat or wrap up

No pressure to talk. Just working in parallel. You in?

Suggested time: [DAY] at [TIME]
Platform: [ZOOM/FACETIME/DISCORD]
```

## Task Initiation Scaffold

> Can't start? Use this framework to break through the wall.

**The task I need to do:** [WRITE IT]

**Step 1: Make it tiny.**
What is the SMALLEST possible first action? (Not "write the report" — "open the document")

```
First physical action: [WRITE IT]
```

**Step 2: Make it time-bound.**
I will do this first action for exactly ___ minutes. (Pick 2, 5, or 10.)

**Step 3: Make it concrete.**
Where will I be sitting? What will be open on my screen? What will my hands be doing?

```
I will be: [WHERE]
I will have open: [WHAT]
My hands will: [DOING WHAT]
```

**Step 4: Remove one barrier.**
What is ONE thing making this harder that I can eliminate right now?

```
Barrier: [WHAT]
How I'll remove it: [HOW]
```

**Step 5: GO.** Set timer. Do the tiny thing. Decide after.
