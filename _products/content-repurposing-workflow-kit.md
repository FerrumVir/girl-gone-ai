# Content Repurposing Workflow Kit

> Complete system for turning 1 piece of content into 10+ platform-ready assets. Includes the multiplication framework, platform-specific templates, batch scheduling workflow, and checklists by content type.

---

## SECTION 1: THE 1-TO-10 CONTENT MULTIPLICATION FRAMEWORK

### The Core Concept
One piece of "pillar content" (long-form) becomes 10+ pieces of platform-specific content.

```
ONE PILLAR CONTENT PIECE
         ↓
┌────────────────────────────────────────────────┐
│ 1. Twitter/X thread (5-10 tweets)              │
│ 2. LinkedIn post (story format)                │
│ 3. LinkedIn carousel (key points)              │
│ 4. Instagram carousel (visual format)          │
│ 5. Instagram Reel (30-60 sec video)            │
│ 6. TikTok video (15-60 sec)                    │
│ 7. Pinterest pin (infographic or tip)          │
│ 8. Email newsletter                            │
│ 9. YouTube Short or full video                 │
│ 10. Quote graphics (3-5 shareable images)      │
│ 11. Blog post (if pillar wasn't a blog)        │
│ 12. Podcast talking points or solo episode     │
└────────────────────────────────────────────────┘
```

### Pillar Content Types (Source Material)
```
TYPE A: Blog Post (1500+ words)
TYPE B: Podcast Episode (20-60 min)
TYPE C: YouTube Video (10-30 min)
TYPE D: Newsletter/Email (500-1000 words)
TYPE E: Live Stream / Webinar (30-90 min)
TYPE F: Thread or Long-Form Social Post (that performed well)
```

## SECTION 2: REPURPOSING BY SOURCE TYPE

### SOURCE: BLOG POST → Everything Else

**Blog → Twitter/X Thread**
```
PROCESS:
1. Extract the 5-8 key points from the blog
2. Turn each into a standalone tweet (under 280 characters)
3. First tweet = hook (the most compelling insight or claim)
4. Last tweet = CTA + link to full blog

TEMPLATE:
Tweet 1: [HOOK — surprising statement, stat, or question]
Tweet 2: [KEY POINT 1 — expanded slightly]
Tweet 3: [KEY POINT 2]
Tweet 4: [KEY POINT 3]
Tweet 5: [KEY POINT 4]
Tweet 6: [KEY POINT 5]
Tweet 7: [SUMMARY — the "so what" takeaway]
Tweet 8: [CTA — "Full breakdown here: [LINK]" or "Follow for more [TOPIC]"]
```

**Blog → LinkedIn Post**
```
PROCESS:
1. Pick the ONE most interesting insight from the blog
2. Open with a hook (1-2 lines that stop the scroll)
3. Tell the story or explain the insight (main body)
4. End with a takeaway + engagement question
5. Do NOT link in the post (kills reach) — put link in first comment

TEMPLATE:
[HOOK — 1-2 lines, creates curiosity]

[BLANK LINE]

[STORY OR CONTEXT — 3-5 short paragraphs]

[THE INSIGHT/LESSON — what did you learn or what should they know]

Here's what I'd recommend:
- [ACTION 1]
- [ACTION 2]
- [ACTION 3]

[ENGAGEMENT QUESTION — "Have you experienced this? What worked for you?"]

---
[CTA — "Full breakdown on my blog — link in comments"]
```

**Blog → Instagram Carousel**
```
PROCESS:
1. Identify 6-8 key points (each becomes one slide)
2. Slide 1 = attention-grabbing title
3. Slides 2-7 = one point per slide (headline + 1-2 sentence explanation)
4. Slide 8 = summary or recap
5. Slide 9 = CTA

SLIDE FORMAT (per slide):
─────────────────────
[NUMBER or ICON]

[HEADLINE — 4-7 words]

[1-2 sentence explanation
that adds context]
─────────────────────
Character limit per slide: keep under 40 words
Font: Large enough to read without zooming
```

**Blog → Instagram Reel/TikTok**
```
PROCESS:
1. Pick the single MOST shareable point from the blog
2. Script a 30-45 second video delivering that one point
3. Open with a hook (first 2 seconds)
4. Deliver the value (middle)
5. Close with CTA

SCRIPT TEMPLATE:
[0-3 sec]: "Stop doing [COMMON THING] — here's why..."
[3-20 sec]: [Explain the one key point conversationally]
[20-27 sec]: "Instead, try [ALTERNATIVE]"
[27-30 sec]: "Follow for more [TOPIC] tips"

Text overlay: Key phrases that reinforce what you're saying
```

**Blog → Pinterest Pin**
```
PROCESS:
1. Create a 1000x1500 pin with the blog title as headline
2. Add 3-5 bullet points (key takeaways visible on pin)
3. Include your website URL
4. Write a keyword-rich pin description (150-300 characters)
5. Link pin directly to blog post

PIN DESCRIPTION TEMPLATE:
"[PRIMARY KEYWORD] — [WHAT THEY'LL LEARN]. This guide covers [POINT 1], [POINT 2], and [POINT 3]. Perfect for [AUDIENCE]. Click to read the full [BLOG/GUIDE]. #[keyword] #[keyword]"
```

**Blog → Email Newsletter**
```
PROCESS:
1. Write a personal intro (why this topic matters to you/them right now)
2. Summarize the blog's key insights (don't paste the whole thing)
3. Add one extra insight NOT in the blog (exclusive for subscribers)
4. Link to full blog for details
5. End with a question or CTA

TEMPLATE:
Subject: [CURIOSITY-DRIVEN SUBJECT LINE]

Hey [NAME],

[1-2 SENTENCES: Personal intro / why you're writing about this]

I just published [BLOG TITLE] and here are the key takeaways:

1. [KEY POINT — one sentence summary]
2. [KEY POINT]
3. [KEY POINT]

But here's something I didn't include in the blog: [BONUS INSIGHT — exclusive for email list]

Full breakdown here: [LINK]

[QUESTION OR CTA]

[YOUR NAME]
```

### SOURCE: PODCAST EPISODE → Everything Else

**Podcast → Blog Post**
```
PROCESS:
1. Transcribe the episode (use Descript, Otter.ai, or Riverside transcript)
2. Identify the 5-7 key sections/topics discussed
3. Create headers for each section
4. Turn conversational language into written language (tighten, remove filler)
5. Add: intro paragraph, subheadings, bullet points, conclusion, SEO optimization
6. Embed the episode audio at the top of the blog post

OUTLINE:
Title: [SEO-OPTIMIZED TITLE — different from episode title]
Intro: 100-word summary of what the episode covered
Section 1: [TOPIC] — key insights from that segment
Section 2: [TOPIC]
Section 3: [TOPIC]
Section 4: [TOPIC]
Key Takeaways: bullet point list
CTA: Listen to the full episode [EMBED/LINK]
```

**Podcast → Audiogram/Video Clips**
```
PROCESS:
1. While editing, timestamp the best 3-5 moments (quotes, insights, funny moments)
2. Cut 30-60 second clips of each
3. Add captions/subtitles (essential — most people watch without sound)
4. Add waveform animation or video if you record video
5. Post as Reels, TikToks, YouTube Shorts, and Twitter video

CLIP SELECTION CRITERIA:
- Does this stand alone without context? (must be self-contained)
- Is there an emotional moment? (laughter, surprise, strong opinion)
- Is there a clear takeaway? (not just context/setup)
- Would someone share this? (quotable, relatable, or useful)
```

**Podcast → Quote Graphics**
```
PROCESS:
1. Pull 3-5 powerful quotes from the episode
2. Design as branded graphics (1080x1080 for IG, 1200x675 for Twitter)
3. Include: quote text, speaker attribution, episode number, your show branding

QUOTE SELECTION:
- Standalone (makes sense without context)
- Quotable (sounds good as a pull-quote)
- Opinionated or insightful (not generic)
- Short enough to fit on a graphic (under 30 words ideal)
```

**Podcast → Twitter/X Thread**
```
PROCESS:
1. Identify the episode's thesis + 5-7 supporting points
2. Tweet 1: "I just had [GUEST] on my podcast and they dropped this insight: [BEST MOMENT]"
3. Tweets 2-6: Key points from the conversation
4. Final tweet: "Full episode here: [LINK]. RT if this was useful."
```

### SOURCE: YOUTUBE VIDEO → Everything Else

**YouTube → Blog Post (Video-to-Article)**
```
PROCESS:
1. Pull transcript (YouTube auto-captions or Descript)
2. Organize into logical sections with subheadings
3. Add screenshots from the video as images
4. Include: "Prefer to watch? [VIDEO EMBED]" at the top
5. Add internal links, CTAs, and SEO optimization
6. Publish (Google ranks both the video and the article)
```

**YouTube → Shorts/Reels/TikTok**
```
PROCESS:
1. Identify 3-5 "highlight moments" (best tips, reactions, results)
2. Cut 15-60 second vertical clips
3. Re-edit for vertical format (zoom/crop if originally 16:9)
4. Add text overlay and captions
5. New hook for each clip (first 2 seconds must grab attention)
6. Post as YouTube Shorts + cross-post to IG Reels and TikTok

CLIPPING STRATEGY:
- Best 1 tip = standalone Short
- A surprising result or reaction = standalone Short
- "Here's the mistake most people make" moment = standalone Short
- Any time you said something quotable/controversial = Short
```

**YouTube → Email Digest**
```
PROCESS:
1. Summarize the video in 3-4 paragraphs
2. Include one key insight + one exclusive bonus point
3. Embed the video or link to it
4. Frame it as: "In case you missed this week's video..."

Good for: weekly recap emails sent to subscribers who prefer reading
```

## SECTION 3: PLATFORM-SPECIFIC REFORMATTING TEMPLATES

### Platform Character Limits & Specs (2026)
```
Twitter/X: 280 characters (or 25,000 for Twitter Blue long-form)
LinkedIn: 3,000 characters per post
Instagram caption: 2,200 characters
TikTok caption: 4,000 characters
Pinterest description: 500 characters
YouTube description: 5,000 characters
Email subject line: 50-60 characters recommended
Facebook: 63,206 characters (but optimal is under 150)
```

### Format Adaptation Quick Reference
```
FROM LONG-FORM (Blog/Podcast/Video) TO:

→ TWITTER THREAD: Extract 5-8 standalone insights. Each under 280 chars. Hook first.

→ LINKEDIN POST: Pick ONE insight. Story format. 1,200-1,800 characters. Hook in first 2 lines (before "see more").

→ INSTAGRAM CAROUSEL: 6-10 key points. One per slide. Under 40 words per slide. Visual.

→ INSTAGRAM REEL: ONE point. 30-45 seconds. Hook in first 2 seconds. Text on screen.

→ TIKTOK: ONE point. 15-60 seconds. Trend format if applicable. Conversational.

→ PINTEREST: Title + 3-5 visible takeaways on pin image. Keyword-rich description. Link to full content.

→ EMAIL: Personal intro + 3-5 key points summarized + link to full piece + exclusive bonus.

→ YOUTUBE SHORT: ONE moment (best tip or reaction). 15-60 seconds. Vertical. Strong hook.

→ QUOTE GRAPHIC: Best one-liner from the content. Branded design. Under 25 words.
```

## SECTION 4: BATCH SCHEDULING WORKFLOW

### The 2-Hour Repurposing Block

```
WHEN: Schedule one 2-hour block per week (e.g., Tuesday morning)
INPUT: One published pillar content piece from this week
OUTPUT: 8-12 platform-ready pieces scheduled across the week

MINUTE-BY-MINUTE WORKFLOW:

0:00 - 0:10 — REVIEW
- Re-read/re-listen to your pillar content
- Highlight or timestamp the best 5-7 moments/points
- Note the single best insight (for short-form)

0:10 - 0:30 — WRITE
- Twitter thread (copy key points, format as tweets)
- LinkedIn post (one insight expanded into story format)
- Email newsletter draft (summary + bonus)
- Instagram caption (for carousel or standalone)

0:30 - 0:50 — DESIGN
- Instagram carousel (slides in Canva — use your template)
- Pinterest pin (1-2 designs for the same content)
- Quote graphics (2-3 shareable images)
- Reel/TikTok cover image (if applicable)

0:50 - 1:10 — RECORD/EDIT
- Film 1-2 short talking-head clips (30-45 seconds each)
- Or: clip 2-3 moments from existing video/audio
- Add captions and text overlay

1:10 - 1:30 — SCHEDULE
- Upload all assets to scheduling tool (Later, Buffer, Planoly)
- Add captions, hashtags, alt text
- Space posts across the week (not all on one day)
- Queue email in your email tool

1:30 - 1:40 — CROSS-LINK
- Add links between pieces (blog links to podcast, email links to video, etc.)
- Pin comment on Reel linking to full content
- Update show notes with new assets

1:40 - 2:00 — DOCUMENT
- Log what you created in your content tracker
- Note performance benchmarks to review next week
- Add ideas for next week's repurposing
```

### Publishing Schedule (Spacing Guide)
```
When your pillar content publishes (e.g., Monday):

MONDAY: Pillar content goes live (blog/podcast/video)
        Email newsletter sent (drives traffic to pillar)

TUESDAY: Twitter thread (teases pillar content insights)
         LinkedIn post (one key takeaway)

WEDNESDAY: Instagram carousel (key points visual format)
           Pinterest pin (drives long-term search traffic)

THURSDAY: Reel or TikTok (best moment as short video)
          Quote graphic on Stories

FRIDAY: Second angle post (different platform, different hook)
        YouTube Short (if source was video)

WEEKEND: Engagement-focused post (question, poll, or reflection)
         Repurposed quote on Stories

Rule: Same content, different format, spaced out. Followers on multiple platforms won't see the same thing on the same day.
```
