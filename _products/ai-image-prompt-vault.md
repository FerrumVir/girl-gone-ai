# AI Image Generation Prompt Vault — 200+ Prompts for Midjourney, DALL-E & Stable Diffusion

---

## How to Use This Vault

This vault contains 210+ image generation prompts organized by visual style and use case. Each prompt is optimized for AI image generators and includes style modifiers, quality boosters, and format specifications.

**Compatible with:**

- Midjourney (v5+)
- DALL-E 3
- Stable Diffusion (SDXL)
- Leonardo AI
- Adobe Firefly

**How prompts are structured:**

```
[Subject/Scene Description], [Style Modifiers], [Lighting], [Mood/Atmosphere], [Technical Specs]
```

**Key to notation:**

- `{SUBJECT}` — replace with your specific subject
- `--ar X:Y` — Midjourney aspect ratio parameter
- `[Negative]` — terms to exclude (use in negative prompt field for SD/Leonardo)
- `Quality boosters` — terms that improve output quality across all platforms

**Quality Boosters (add to any prompt):**

- `highly detailed, 8k resolution, professional photography`
- `masterpiece, best quality, ultra-realistic`
- `award-winning photography, sharp focus, intricate details`
- `cinematic quality, color graded, high dynamic range`

**Universal Negative Prompts (for Stable Diffusion):**

```
blurry, low quality, distorted, deformed, ugly, bad anatomy, bad proportions, extra limbs, watermark, text, logo, signature, cropped, out of frame, worst quality, low resolution, jpeg artifacts
```
