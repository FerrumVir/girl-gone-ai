The first step to organizing your digital life is knowing exactly what you have. Use this database to catalog every app, account, and service you use.

**Categories to use:** Social Media, Streaming, Productivity, Finance, Shopping, Cloud Storage, Communication, Health/Fitness, News/Media, Education, Gaming, Utilities, Other

**Status options:**
- **Active** — You use this regularly (at least monthly)
- **Occasional** — You use this a few times per year
- **Dormant** — You haven't used it in 3+ months but the account exists
- **To Cancel** — You've decided to get rid of this
- **Cancelled** — Done, account closed

**Importance ratings:**
- **Essential** — You need this for work or daily life
- **Valuable** — Makes your life better but you could survive without it
- **Nice to Have** — You enjoy it but rarely think about it
- **Unnecessary** — No clear reason you're still paying for or using this

### How to Complete This Section

1. **Check your email** for account creation confirmations and welcome emails. Search for terms like "welcome," "verify your email," "your account," and "subscription."
2. **Check your phone** — go through every app and add it to the inventory.
3. **Check your browser** — review saved passwords, bookmarks, and autofill suggestions.
4. **Check your bank/credit card statements** for the past 3 months — look for any recurring digital charges.
5. **Check password managers** if you use one — export your list of saved sites.

### Subscription Decision Framework

For each subscription, answer these questions:

1. **When did I last use this?** If more than 30 days ago, it's a candidate for cancellation.
2. **Could I get this for free?** Many paid tools have free alternatives that are good enough.
3. **Am I on the right plan?** You might be paying for a premium tier when the basic plan covers your actual usage.
4. **Is this a duplicate?** Do you have two apps that do the same thing?
5. **Does this spark genuine value?** Not "I might use it someday" — does it actively improve your life right now?

## Section 3: File Organization System

### The Universal Folder Structure

Use this structure across ALL cloud platforms and local storage. Consistency is the goal.

```
/Documents
    /Finance
        /Taxes
            /2024
            /2025
            /2026
        /Banking
        /Insurance
        /Receipts
    /Work
        /[Company or Client Name]
            /Projects
            /Contracts
            /Reports
    /Personal
        /Medical
        /Legal
        /Education
        /Certificates
    /Home
        /Lease-Mortgage
        /Utilities
        /Warranties
        /Manuals

/Projects
    /[Project Name]
        /Assets
        /Drafts
        /Final
        /Reference

/Media
    /Photos
        /[Year]
            /[Month] or [Event Name]
    /Videos
    /Music

/Archive
    /[Year]
        (Old projects, expired documents, etc.)
```

### File Naming Convention

Use this format for consistent, searchable file names:

**`YYYY-MM-DD_Category_Description_Version`**

Examples:
- `2026-04-09_Tax_W2-Employer-Name_Final`
- `2026-03-15_Project_Client-Proposal_v2`
- `2025-12-01_Medical_Annual-Physical-Results`

Rules:
- Use hyphens (-) instead of spaces
- Use underscores (_) to separate the date, category, description, and version
- Always lead with the date for automatic chronological sorting
- Use "Final" for completed documents, "v1/v2/v3" for working drafts

### Cloud Consolidation Checklist

- [ ] Choose ONE primary cloud platform (Google Drive, Dropbox, iCloud, or OneDrive)
- [ ] Create the universal folder structure on your primary platform
- [ ] Migrate files from secondary platforms to primary platform
- [ ] Download any critical files to a local backup before deleting from old platforms
- [ ] Delete duplicate files across platforms
- [ ] Empty the Downloads folder — process every file (move, rename, or delete)
- [ ] Uninstall cloud apps you're no longer using

### The "Digital Junk Drawer" Triage

For your Downloads folder and Desktop, process each file with this decision:

1. **Is this a duplicate?** Delete it.
2. **Do I need this?** Move it to the correct folder using the naming convention.
3. **Is this temporary/one-time?** (Installer files, shipping labels, etc.) Delete it.
4. **Am I keeping this "just in case"?** Move it to /Archive/[Year]. If you don't touch it for 12 months, delete it.

## Section 4: Email Cleanup Workflow

### Phase 1: The Unsubscribe Blitz (Day 1 — 30 minutes)

1. Search your inbox for "unsubscribe" — this surfaces all marketing/newsletter emails.
2. For each sender, decide: **Read regularly** / **Never read** / **Occasionally useful**
3. Unsubscribe from everything in the "Never read" category.
4. For "Occasionally useful," consider creating a filter to skip the inbox and go to a "Newsletters" folder — you can check it when you want to, not when they want you to.

**Tip:** Services like Unroll.me can speed this up, but manually unsubscribing gives you more control and ensures you're actually removed from lists.

### Phase 2: Folder/Label Architecture (Day 2 — 20 minutes)

Create these folders/labels in your email client:

| Folder | Purpose |
|---|---|
| **@Action** | Emails that require you to DO something (respond, pay, fill out, etc.) |
| **@Waiting** | Things you're waiting on from other people |
| **@Reference** | Emails you need for reference but no action required |
| **Receipts** | Purchase confirmations and receipts |
| **Finance** | Bank alerts, investment updates, tax documents |
| **Newsletters** | Newsletters you actually want to read (on your schedule) |
| **Archive** | Everything else worth keeping |

### Phase 3: Auto-Filter Rules (Day 3 — 20 minutes)

Set up these automatic filters:

| Trigger | Action |
|---|---|
| From bank/financial institutions | Label "Finance," skip inbox |
| Subject contains "receipt" or "order confirmation" | Label "Receipts," skip inbox |
| From newsletter senders you kept | Label "Newsletters," skip inbox |
| From social media notifications | Archive automatically (or delete) |
| From specific important contacts (boss, spouse, etc.) | Star automatically |

### Phase 4: The Purge (Days 4-7 — 20 minutes/day)

1. **Sort by sender** and bulk-delete/archive emails from senders you've unsubscribed from.
2. **Sort by date** and archive everything older than 30 days that doesn't need action.
3. **Search for large attachments** and download important ones to your file system, then delete the emails.
4. **Empty trash and spam** folders.

### Phase 5: Daily Maintenance Routine (Ongoing — 10 minutes/day)

Process your inbox at set times (e.g., 9 AM, 1 PM, 5 PM):

1. **Delete** — spam, irrelevant, unneeded
2. **Do** — if it takes less than 2 minutes, handle it now
3. **Delegate** — forward it and move to @Waiting
4. **Defer** — move to @Action and add a due date/reminder
5. **File** — move to the appropriate folder

## Section 5: Password Audit Checklist

### Step 1: Assess Your Current Situation

- [ ] Do you use a password manager? (If yes, skip to Step 3)
- [ ] Do you reuse the same password across multiple sites?
- [ ] Do you use variations of the same base password?
- [ ] Have you been notified of any data breaches? (Check at haveibeenpwned.com)
- [ ] How many accounts do you have without two-factor authentication (2FA)?

### Step 2: Set Up a Password Manager

Choose one:

| Manager | Free Plan | Paid Plan | Best For |
|---|---|---|---|
| Bitwarden | Unlimited passwords | $10/year | Best free option, open source |
| 1Password | No free plan | $36/year | Families, best UI |
| Dashlane | 25 passwords | $60/year | Extra features (VPN, dark web monitoring) |
| Apple Keychain | Free (Apple only) | Free | Apple ecosystem users |

**Migration process:**
1. Install your chosen password manager on all devices.
2. Import passwords from your browser(s).
3. Go through each account and generate a new, unique password using the manager.
4. Prioritize: financial accounts, email accounts, social media, then everything else.
5. Disable browser password saving once the manager is set up.

### Step 3: Password Strength Audit

Go through your password manager (or list) and flag any passwords that are:

- [ ] Shorter than 12 characters
- [ ] Used on more than one site
- [ ] Based on dictionary words without modification
- [ ] Based on personal information (birthdays, pet names, addresses)
- [ ] Older than 12 months on high-security accounts (banking, email)

### Step 4: Enable Two-Factor Authentication (2FA)

Priority order:
1. [ ] Email accounts (this is the master key to everything else)
2. [ ] Financial accounts (banking, investments, crypto)
3. [ ] Social media accounts
4. [ ] Cloud storage accounts
5. [ ] Shopping accounts (Amazon, etc.)
6. [ ] Work/productivity accounts

**Use an authenticator app** (Google Authenticator, Authy, or the one built into your password manager) over SMS when possible. SMS-based 2FA is better than nothing but can be compromised via SIM swapping.

### Step 5: Secure Your Recovery

- [ ] Save backup codes for all 2FA-enabled accounts in your password manager
- [ ] Set up a recovery email that is different from your primary email
- [ ] Record your password manager's master password on paper and store it in a physically secure location (safe, lockbox)

## Section 7: Maintenance Calendar

### Weekly (Every Sunday — 15 minutes)

- [ ] Process inbox to zero (or close to it)
- [ ] Clear Downloads folder
- [ ] Review @Action and @Waiting email folders
- [ ] Delete unused screenshots and temp files from phone and desktop

### Monthly (1st of each month — 30 minutes)

- [ ] Retake Digital Wellness Score
- [ ] Review subscription charges on bank statement
- [ ] Delete unused apps from phone and computer
- [ ] Check for and install software updates
- [ ] Review and clean up cloud storage (delete old shared files, empty trash)

### Quarterly (Jan, Apr, Jul, Oct — 45 minutes)

- [ ] Full subscription audit (re-evaluate every subscription)
- [ ] Password audit (change any flagged passwords, review 2FA)
- [ ] Archive old projects and files
- [ ] Review and update auto-filters in email
- [ ] Clean up social media (unfollow/mute, review privacy settings, delete old posts if desired)

### Annual (January — 60 minutes)

- [ ] Major file archive (move prior year's completed projects to Archive)
- [ ] Tax document organization (gather W-2s, 1099s, receipts)
- [ ] Full account inventory refresh
- [ ] Review and update emergency access info (can someone you trust access critical accounts if needed?)
- [ ] Evaluate whether your tools are still the right ones (email provider, cloud storage, password manager)
