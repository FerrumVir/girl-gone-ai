## Store Setup Checklist

### Phase 1: Account & Foundation (Day 1)

**Shopify Account:**
- [ ] Sign up at shopify.com (14-day free trial or current offer)
- [ ] Choose your plan (Basic is fine to start — upgrade later)
- [ ] Set your store name (matches your brand — can be changed later)
- [ ] Add your primary email address
- [ ] Set timezone and currency
- [ ] Set your store address (required for taxes and shipping)

**Domain Setup:**
- [ ] Option A: Buy domain through Shopify (simplest)
- [ ] Option B: Connect existing domain (Namecheap, GoDaddy, Google Domains)
- [ ] Set up SSL (automatic with Shopify — verify the green padlock shows)
- [ ] Redirect www to non-www (or vice versa — pick one and stick with it)
- [ ] Set up email forwarding (info@yourdomain.com → your email)

**Essential Settings:**
- [ ] Store details: name, address, phone, email
- [ ] Currency and units: set primary currency, weight units
- [ ] Checkout settings: customer accounts (optional or required), form fields
- [ ] Notifications: review all customer notification emails (order confirmation, shipping, etc.)
- [ ] Files: upload your logo in multiple sizes (favicon, header, social share)

### Product Photography Standards

| Image # | Type | Purpose |
|---------|------|---------|
| 1 | Hero/Main (white background) | Clear product view, used in grids/feeds |
| 2 | Alternate angle (white background) | Show different perspective |
| 3 | Detail/close-up | Show quality, texture, features |
| 4 | Lifestyle (in use/in context) | Help customer visualize owning it |
| 5 | Scale/size reference | Show size relative to common objects |
| 6 (optional) | Packaging | Show what arrives (unboxing appeal) |
| 7 (optional) | Group/collection | Show color variants or related items |

**Image Specifications:**
- Format: JPG or PNG
- Size: 2048 x 2048 px (square — works best across Shopify themes)
- File size: Under 2MB per image (for fast loading)
- Background: White (#FFFFFF) for main images; lifestyle for supplementary
- Consistency: Same lighting, same background, same style across all products

## Shipping & Payment Configuration

### Shipping Setup

**Shipping Strategy Decision:**

| Strategy | Best For | Setup |
|----------|----------|-------|
| Free shipping (built into price) | Higher margins, $30+ products | Set all rates to $0, increase product prices |
| Free shipping over $X threshold | Encouraging larger orders | Create "Free over $X" rate + paid rate below |
| Flat rate shipping | Simplicity, predictable costs | One rate for domestic, one for international |
| Calculated rates (carrier) | Accuracy, heavy/large items | Connect carrier accounts (USPS, UPS, FedEx) |
| Local pickup/delivery | Local businesses | Enable local delivery settings |

---

**Shipping Rate Setup Template:**

| Zone | Rate Name | Condition | Price |
|------|-----------|-----------|-------|
| Domestic | Standard | All orders | $[X] or Free |
| Domestic | Free Shipping | Orders over $[X] | Free |
| Domestic | Express | All orders | $[X] |
| International | Standard International | All orders | $[X] |
| International | Free International | Orders over $[X] | Free |

---

**Shipping Settings Checklist:**
- [ ] Shipping zones created (Domestic, International — or country-specific)
- [ ] Rates set for each zone
- [ ] Product weights entered (for calculated shipping)
- [ ] Packaging dimensions set (for calculated shipping)
- [ ] Shipping policy page written (see Essential Pages below)
- [ ] Order tracking enabled (connect carrier accounts)
- [ ] Fulfillment service selected (self-fulfillment, 3PL, or dropship)
- [ ] Customs info added (for international: HS codes, country of origin)

---

### Payment Setup

**Payment Gateways:**
- [ ] Shopify Payments activated (lowest fees, simplest setup)
- [ ] PayPal connected (some customers strongly prefer PayPal)
- [ ] Shop Pay enabled (one-click checkout — significantly increases conversion)
- [ ] Apple Pay and Google Pay enabled (one-tap mobile purchases)
- [ ] Buy Now Pay Later option (Shop Pay Installments, Afterpay, or Klarna)

**Checkout Optimization:**
- [ ] Express checkout buttons enabled (Shop Pay, Apple Pay, Google Pay above the fold)
- [ ] Customer accounts set to "optional" (don't force account creation)
- [ ] Marketing consent checkbox added (for email collection at checkout)
- [ ] Order notes enabled (if customers need to add instructions)
- [ ] Tipping disabled (unless service-based business)
- [ ] Automatic discounts applied at checkout (if running promotions)

## SEO Setup & Optimization

### Technical SEO Checklist

- [ ] Custom domain connected (not yourstore.myshopify.com)
- [ ] SSL active (green padlock / https://)
- [ ] Sitemap submitted to Google Search Console (yoursite.com/sitemap.xml)
- [ ] Google Analytics 4 connected
- [ ] Meta titles set for all pages (under 60 characters)
- [ ] Meta descriptions set for all pages (under 155 characters)
- [ ] Image alt text added to all product images (descriptive, keyword-rich)
- [ ] URL handles cleaned up (short, keyword-rich, no auto-generated strings)
- [ ] 301 redirects set for any changed URLs
- [ ] Page speed optimized (compress images, minimize apps, use system fonts)

---

### SEO Template for Product Pages

```
PAGE TITLE (under 60 characters):

Example: "Organic Cotton Oversized Tee - Sage Green | Everloom"

H1 TAG (product title):
[Your product title serves as H1 — make it keyword-rich but natural]
```

## Post-Launch Optimization

### Week 1-2: Monitor & Fix

**Daily Checks:**
- [ ] Orders processing correctly
- [ ] Payments clearing
- [ ] Emails sending (confirmation, shipping, etc.)
- [ ] Website loading fast (test on mobile)
- [ ] Customer questions answered within 24 hours

**Analytics to Watch:**

| Metric | Tool | Action If Low |
|--------|------|---------------|
| Traffic | Google Analytics | Increase marketing spend/effort |
| Bounce rate | Google Analytics | Improve landing pages/loading speed |
| Add-to-cart rate | Shopify Analytics | Improve product pages/pricing |
| Cart abandonment | Shopify Analytics | Set up abandoned cart emails |
| Conversion rate | Shopify Analytics | Improve checkout/trust signals |
| Average order value | Shopify Analytics | Add upsells, bundles, threshold shipping |

---

### Month 1 Growth Actions

- [ ] Set up abandoned cart email sequence (recovers 5-15% of lost sales)
- [ ] Install product review app (social proof increases conversion 15-30%)
- [ ] Add upsell/cross-sell suggestions on product and cart pages
- [ ] Create a referral or loyalty program
- [ ] Start collecting UGC (user-generated content) from customers
- [ ] Test one paid advertising channel with small budget ($10-20/day)
- [ ] Analyze top-performing products and double down on marketing them
- [ ] Survey first customers: "How did you find us? What almost stopped you from buying?"
